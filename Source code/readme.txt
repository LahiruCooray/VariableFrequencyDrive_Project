Instructions for Compilation
If you are compiling this code using AVR GCC 8-bit compiler, please follow these steps:

Include the Modified Display Library:
    Ensure that you have added the modified display library provided in this repository. The library is crucial for correct functionality of the displays used in this project.

Setting Up the Environment:
    Make sure AVR GCC 8-bit compiler is properly installed on your system.

Compiling the Code:
    Use your preferred IDE or command line tools compatible with AVR GCC to compile the code.
